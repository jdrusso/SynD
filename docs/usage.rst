==================
Installation/Usage
==================


To install :code:`synd`:

.. code-block:: bash

    cd </path/to/synd>

    conda env create -f environment.yml

or, to install it into an existing conda environment:

.. code-block:: bash

    cd </path/to/synd>

    conda env update --name <your environment> --file environment.yml
